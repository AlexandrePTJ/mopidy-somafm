mopidy-somafm
=============

`Mopidy <http://www.mopidy.com/>`_ extension for playing music from
`SomaFM <http://somafm.com/>`_.

Usage
-----

Install by running::

    sudo pip install Mopidy-SomaFM

This extension requires activation of 'stream' extension.
